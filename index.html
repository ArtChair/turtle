Модуль Turtle

Turtle это модуль для Питон, позволяющий создавать графические объекты, риснуки в специальном окне. Модуль Turtle можно использовать для создания игр на Питоне.

Для работы с исполнителем Черепаха сначала нужно подключить модуль Turtle

from turtle import *

На самом деле у этой команды более широкий функционал. Эта команда выставляет Черепаху в начальное положение, сбрасывает все параметры на значения по умолчанию и стирает все, что было нарисовано до этого.

Теперь, после запуска программы мы увидим окно «Python Turtle Graphics». В центре окна будет отображаться перо в виде стрелки, повернутой вправо. Это начальное положение исполнителя «Черепаха». Координаты точки (0,0). При рисовании используется Декартова система координат.

Список команд исполнителя для рисования

up() – поднять перо. Используется в случае, если необходимо переместить Черепаху без оставления следа рисования.

up()

down() – опустить перо. Используется для того, чтобы при перемещении Черепаха оставляла за собой след (в виде линии).

down()

goto(x, y) – перемещение пера в точку с координатами (x,y).

goto(100,75)

color(цвет1,[цвет2]) – цвет пера и заливки. Если указан только один цвет, то цвет пера и цвет заливки совпадают, а если 2 цвета через запятую, то первый – это цвет пера, а второй – цвет заливки. Допустимы два способа задания цвета: словесное название цвета и шестнадцатиричное обозначение цвета. При шестнадцатиричном обозначении в начало добавляется символ «#».

color('red') # задали красный цвет пера
color('blue') # задали синий цвет пера

forward(шаг) перемещение вперед. Перемещает Черепаху вперед по аправлению стрелки или туловища Черепахи на количество точек, указанное в параметре шаг. Шаг может быть как целым числом, так и дробным.

forward(50) # перемещение на 50 точек вперед

backward(шаг) – перемещение назад. Перемещает Черепаху в направлении, противоположном направлению стрелки или туловища Черепахи на количество точек, указанное в параметре шаг.

backward(50) # перемещаем на 50 точек назад

right(угол) – поворот вправо. Поворачивает исполнителя вправо (по часовой стрелке) относительно направления его движения на значение, указанное в параметре угол. Угол по умолчанию измеряется в градусах.

right(90) # поворот на 90 градусов по часовой стрелке

left(угол)– поворот влево. Поворачивает исполнителя влево (против часовой стрелки) относительно направления его движения на значение, указанное в параметре угол. Угол по умолчанию измеряется в градусах.

left (90) # поворот на 90 градусов против часовой стрелки

dot([size],[color])– рисует точку в текущей позиции Черепахи. Если параметры не указаны, то размер точки равен 1, цвет черный. Иначе размер и цвет берутся из параметров.

dot(20, 'red') # красный круг с радиусом - 20

circle(r)– рисование окружности. Рисование окружности радиусом r из текущей позиции исполнителя. Радиус может быть как положительным, так и отрицательным числом. Если радиус положительный, то окружность рисуется против часовой стрелки, а если отрицательный, то по часовой стрелке.

circle(-50) # рисуем окружность радиуса 50 по часовой стрелке

begin_fill() ... end_fill() – включение и выключение закрашивания области.

begin_fill() # включили режим закрашивания
end_fill() # выключили режим закрашивания

tracer(f, [delay]) – отображение процесса рисования.

tracer(0) # отключили режим трассировки закрашивания
tracer(1) # включили режим трассировки
